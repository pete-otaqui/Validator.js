Validator.js by pete-otaqui

A simple Javascript validation class that does not require a <form> tag

Validator.js let's you quickly validate javascript variables, or create complex validation chains with custom error messages. It is designed to work independently of the DOM, so is perfect for Backbone's validate() method, or can easily be modified for use as a CommonJS module

It's unusual for a project to pride itself on what it doesn't do, but in the face of data validation in Javascript, not being tied to a form and input fields is something of a rarity, and also highly desirable if you want to use something like Backbone or NodeJS.

See the blog post, the project page or the API Documentation for more.
